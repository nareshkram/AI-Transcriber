# AI-Transcriber  

AI-Transcriber एक ऑडियो-टू-टेक्स्ट ट्रांसक्रिप्शन टूल है जो AI तकनीक का उपयोग करता है।  

## फीचर्स  
✅ ऑडियो से टेक्स्ट कन्वर्जन  
✅ मल्टी-फॉर्मेट सपोर्ट (MP3, WAV, आदि)  
✅ स्पीच-टू-टेक्स्ट API का उपयोग  
✅ आसान और यूजर-फ्रेंडली UI
